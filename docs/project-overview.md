# Project Overview

## Vision
Le produit est un assistant pour Donjon & Dragons . 
Il est destiné à streamline la création de rencontre et les combats.
Il s'appuie sur les API disponible pour les monstres et les règles officielles tout en autorisant l'utilisation de homebrew.
L'objectif est d'améliorer les produits existants tels que "Improved Initiative" ou ShieldMaiden parfois trop confus en terme d'utilisation.
Les phases de combat disposeront d'un accès à une narration d'ambiance via l'utilisation de l'IA.
Les informations sur les PC sont tracés pour pouvoir enchainer les rencontres sans perdre le compte des pv, nombre de sort et autres.
Le produit est fait pour une utilisations dans le cadre d'une partie en TTRPG, donc on doit pouvoir laisser les lancer de dès aux joueurs et DM dans la majorité des cas et simplement en traiter les résultats.

## Target users
Les DM de D&D 5.5e

## Product goals
- Construire un Encounter Builder permettant de visualiser et stocker des combats d'une partie pour les jouer.
- Implémenter les règles de combat de D&D pour streamline les combats de manière efficace
- Fournir une interface visuellement agréable pour l'usage long.
- Permettre de stocker les informations joueurs pour tracker à terme d'autre infos comme les emplacements de sort, les pv actuel...

## Non-goals
- Jouer une partie complète de D&D
- Faire des combats en full automatique
